# User Dashboard

This is a user dashboard application built using the MERN stack.

## Installation

To get started with the application, run the following command in your terminal:

```bash
npm install
```

## Start the Application

To start the application, run the following command in your terminal:

```bash
npm start
```

This will start the application on port 3000.

## API Endpoints

The user dashboard application uses the following API endpoints:

* `GET /users`: Retrieves a list of all users.
* `GET /users/:id`: Retrieves a single user by ID.
* `POST /users`: Creates a new user.
* `PUT /users/:id`: Updates an existing user.
* `DELETE /users/:id`: Deletes a user.

## Database

The user dashboard application uses a MongoDB database.

## Tests

To run tests for the application, use the following command in your terminal:

```bash
npm test
```
